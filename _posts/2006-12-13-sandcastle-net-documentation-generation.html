I have been using Sandcastle to generate documentation for .NET libraries at work since the demise of the NDoc project.



Currently Sandcastle is configured via a myriad of XML file tweaks. Sandcastle Help File Builder is a nice, NDoc-style GUI that takes some of the configuration hassle away.
